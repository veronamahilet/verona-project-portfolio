# myLexiMap

myLexiMap is an in-progress JavaFX project focused on text prediction and data structure analysis. The application is designed to train on user-provided text, generate prediction support for sentence building, and compare the performance of multiple data structures.

## Current Status
This project is currently in development. The present version focuses on the JavaFX interface structure, layout design, and preparation for the machine workflow.

## Planned Goals
- Train the system using user-provided text
- Support text prediction during sentence building
- Compare model behavior using different data structures
- Display performance and storage statistics in the interface

## Data Structures Planned
- List-based model
- Binary Search Tree (BST) model
- HashMap-based model

## Current Features
- JavaFX multi-section interface
- GridPane-based root layout
- Dedicated panels for performance, data overview, and analysis
- Suggestion display area
- Input area with submit, clear, and done controls
- Initial application stage design for training and prediction flow

## Technologies
- Java
- JavaFX
- CSS
