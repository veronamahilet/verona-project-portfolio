# Weather App (API Integration)

A web application that retrieves and displays real-time weather data based on a user-provided zip code using an external API.

## Features
- Fetch weather data using OpenWeather API
- Display city name, condition, temperature, humidity, wind speed, and pressure
- Dynamic UI update based on API response
- Smooth animation when displaying results
- Error handling for invalid input

## Technologies
- HTML
- CSS
- JavaScript
- jQuery
- OpenWeather API

## Concepts Practiced
- API integration (AJAX requests)
- JSON data handling
- DOM manipulation
- Event handling
- UI animation with CSS
