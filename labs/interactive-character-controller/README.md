# Interactive Character Controller

A small interactive application that allows a character to appear, move, jump, and change states using button controls.

## Features
- Toggle character visibility
- Move left and right within boundaries
- Jump animation using CSS transitions
- Switch between walking and standing states

## Technologies
- HTML
- CSS
- JavaScript

## Concepts Practiced
- DOM manipulation (create/remove elements)
- Event handling
- State management
- CSS transitions and animations
