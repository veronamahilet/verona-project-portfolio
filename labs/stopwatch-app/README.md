# Stopwatch App

A digital stopwatch built using HTML, CSS, and JavaScript, allowing users to track time with interactive controls.

## Features
- Start, stop, and reset functionality
- Time tracking (hours, minutes, seconds, tenths)
- Real-time display updates
- Prevention of multiple timers running simultaneously

## Technologies
- HTML
- CSS
- JavaScript

## Concepts Practiced
- setInterval and clearInterval
- DOM manipulation
- Event handling
- State management
